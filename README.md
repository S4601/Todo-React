
# React Todo App

A robust and user-friendly Todo application designed for efficient task management. The app features an intuitive interface divided into two main workspaces: Uncompleted Todos and Completed Todos. Users can easily keep track of their tasks with built-in functionalities for advanced filtering, dynamic sorting, and seamlessly moving items between the active and completed sections.

## ✨ Key features
-  **Task Management**: Easily toggle tasks between "Uncompleted" and "Completed" states with a single click.

- **Advanced Filtering**: Quickly find specific tasks (e.g., by user) without reloading the page.

- **Dynamic Sorting**: Organize your lists by title (A-Z/Z-A), date, or default order.

- **Pagination**: Efficiently handle large lists of tasks by dividing them into easy-to-navigate pages.

- **Responsive Design**: Clean and structured UI that works seamlessly across devices.

## Built With

* **[React](https://react.dev/)** - Core frontend library for building the UI.
* **[TypeScript](https://www.typescriptlang.org/)** - For strict type checking, robust code, and fewer runtime errors.
* **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid and responsive styling.

* **[Fetch API](https://jsonplaceholder.typicode.com)** - Used for fetching initial mock data (Users and Todos) from `https://jsonplaceholder.typicode.com`.

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

#### Clone the repository

```sh
   git clone https://github.com/S4601/Todo-React.git
```
#### Navigate into the project directory

```sh
    cd Todo-React
```
#### Install dependencies.

```
npm install
```

#### Building

To create a production version of your app:

```
npm run build
```

You can preview the production build with `npm run preview`.
